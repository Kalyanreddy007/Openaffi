# OpenAffi CRM

Modern Sales CRM built for sales teams and agencies.

## Features
- Authentication
- Dashboard
- Companies (Accounts)
- Contacts
- Leads
- Opportunities
- Tasks & Activities
- Reports & Analytics
- Role-Based Access Control

## Tech Stack
### Frontend
- Next.js
- TypeScript
- Tailwind CSS

### Backend
- FastAPI
- SQLAlchemy
- JWT Authentication

### Database
- PostgreSQL

### Infrastructure
- Docker
- Docker Compose
